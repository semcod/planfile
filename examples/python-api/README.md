# Python Library API Examples

Examples of using planfile as a Python library in your applications.

## Files

- `01_basic_usage.py` - Basic Planfile class usage
- `02_ticket_management.py` - CRUD operations on tickets
- `03_integration.py` - Integrating planfile into existing tools
- `04_bulk_operations.py` - Bulk ticket operations
- `05_filtering.py` - Advanced ticket filtering

## Running Examples

```bash
# Navigate to examples directory
cd examples/python-api

# Run individual examples
python 01_basic_usage.py
python 02_ticket_management.py

# Or run all
./run_all.sh
```

## Prerequisites

```bash
pip install planfile
```
